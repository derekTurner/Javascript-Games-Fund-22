# HTML5 introduction

HTML5 is a markup language used to structure a document.



```html
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>First Level Heading</h1>
        <p>Opening paragraph.</p>
    </body>
</html>
```
